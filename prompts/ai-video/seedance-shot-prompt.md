# Seedance 单镜头提示词资产

## 资产信息

| 字段 | 内容 |
|------|------|
| 类型 | AI 视频 |
| 适用场景 | 将单个分镜转成 Seedance 2.0 视频生成提示词 |
| 适用模型/平台 | Seedance 2.0 / 其他图生视频或文生视频工具 |
| 输入变量 | `{角色设定}`、`{场景}`、`{镜头动作}`、`{情绪}`、`{连续性要求}` |
| 输出结果 | 单镜头视频 prompt |
| 标签 | Seedance, AI视频, 分镜, 镜头 |
| 版本 | v1.0 |

## 标准版提示词

```text
请根据下面的分镜信息，生成一个适合 Seedance 2.0 的单镜头视频提示词。

角色设定：{角色设定}
场景：{场景}
镜头动作：{镜头动作}
人物动作：{人物动作}
情绪：{情绪}
光线：{光线}
连续性要求：{连续性要求}

要求：
- 只描述一个镜头
- 保持角色外貌、服装、道具一致
- 镜头运动要清楚
- 不要加入未提到的新角色或新场景
- 输出中文提示词和英文提示词
```

## 增强版提示词

```text
你是一名 AI 视频分镜导演。请把以下分镜转写成 Seedance 2.0 单镜头 prompt。

必须包含：
1. 主体与角色一致性
2. 场景与时间
3. 镜头景别
4. 摄影机运动
5. 人物动作
6. 光线色彩
7. 情绪氛围
8. 连续性要求
9. 避免事项

输入：
角色设定：{角色设定}
场景：{场景}
镜头动作：{镜头动作}
人物动作：{人物动作}
情绪：{情绪}
光线：{光线}
连续性要求：{连续性要求}

输出格式：
## 中文提示词
## English Prompt
## 生成风险
```

## 使用建议

适合在完整分镜确认后逐镜生成。不要一次把多个镜头塞进一个 prompt，容易导致角色和场景混乱。
