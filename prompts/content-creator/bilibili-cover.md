# B站 AI 教程封面提示词

## 资产信息

| 字段 | 内容 |
|------|------|
| 类型 | 内容创作 / AI 生图 |
| 适用场景 | 为 AI 工作流、ComfyUI、AI 视频教程生成封面方向 |
| 适用模型/平台 | ChatGPT / FLUX / Midjourney / 即梦 / 其他生图工具 |
| 输入变量 | `{视频主题}`、`{核心痛点}`、`{视觉主体}`、`{风格}` |
| 输出结果 | 封面图生成提示词 + 封面文案建议 |
| 标签 | B站, 封面, AI教程, ComfyUI |
| 版本 | v1.0 |

## 标准版提示词

```text
请为一个 B站 AI 教程视频设计封面图提示词。

视频主题：{视频主题}
核心痛点：{核心痛点}
视觉主体：{视觉主体}
目标观众：AI 生图和 ComfyUI 新手
风格：清晰、科技感、教程感，有明显视觉焦点，不要过度花哨

请输出：
1. 封面主视觉描述
2. 中文标题文案 3 个
3. AI 生图提示词
4. 不建议出现的元素
```

## 增强版提示词

```text
你是一名 B站科技区封面设计师。请围绕 `{视频主题}` 设计一张适合 AI 工作流教程的封面。

要求：
- 第一眼能看出这是 AI / ComfyUI / 工作流相关内容
- 画面主体明确，不要堆太多元素
- 标题文案控制在 8-14 个中文字符
- 适合 16:9 横版封面
- 背景可以有节点、流程线、模型卡片、云端平台等元素
- 避免廉价赛博朋克、过度霓虹、杂乱 UI 和错别字

请输出：
## 封面定位
## 主视觉方案
## 标题文案
## 生图提示词
## 负面提示词
## 后期排版建议
```

## 使用建议

适合用于 B站视频发布前的封面头脑风暴。正式封面里的中文标题建议后期人工排版，不建议完全依赖生图模型生成文字。
