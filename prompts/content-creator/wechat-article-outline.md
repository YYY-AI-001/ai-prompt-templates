# 公众号文章大纲提示词

## 资产信息

| 字段 | 内容 |
|------|------|
| 类型 | 内容创作 |
| 适用场景 | 将 B站视频选题扩展为公众号图文 |
| 适用模型/平台 | ChatGPT / Claude / Gemini |
| 输入变量 | `{文章主题}`、`{目标读者}`、`{核心观点}`、`{承接链接}` |
| 输出结果 | 公众号文章标题、大纲、开头和结尾 |
| 标签 | 公众号, 图文教程, 内容复用 |
| 版本 | v1.0 |

## 标准版提示词

```text
请把这个视频选题扩展成一篇适合公众号阅读的图文大纲。

文章主题：{文章主题}
目标读者：{目标读者}
核心观点：{核心观点}
承接链接：{承接链接}

要求：
- 用新手能理解的语言
- 不要堆术语
- 每个小节只解释一个问题
- 结尾自然引导关注后续系列

请输出：
1. 标题 5 个
2. 文章开头
3. 正文大纲
4. 每节要点
5. 结尾引导
```

## 增强版提示词

```text
你是「源力魔方」的公众号编辑，擅长把 AI 工作流视频内容整理成清楚、耐读、适合新手收藏的图文教程。

请基于 `{文章主题}` 写一份公众号文章结构。

目标读者：{目标读者}
核心观点：{核心观点}
内容定位：AI 工作流扫盲 / ComfyUI 实操 / AI 视频分镜 / 模型观察

写作要求：
- 先用真实困惑开场
- 第二段给出明确判断
- 正文分 4-6 个小节
- 避免夸张标题党
- 每节都有一句可被读者记住的判断
- 结尾承接 B站、GitHub、RunningHub 或知识星球

请按 Markdown 输出完整大纲。
```

## 使用建议

适合把短视频脚本转成公众号文章，不适合直接生成最终长文。建议先用这个生成大纲，再逐段展开。
