# Qwen-Image Chinese Text Poster

适用模型：Qwen-Image、Qwen-Image-2512、GLM-Image、ERNIE-Image

## Prompt

```text
一张干净现代的中文科技海报，主题是“AI模型入门”。画面中心是由多个发光模块连接成的抽象结构，背景为深灰到黑色渐变，带有细腻的蓝白色科技线条。海报顶部清晰显示中文标题：“AI模型入门”。标题字体简洁有力，字形端正，不能出现错别字。下方留出副标题区域，但不要生成小字正文。整体风格像高质量科技课程封面，构图居中，层级清楚，适合竖版封面图。
```

## Negative Prompt

```text
错别字，乱码，重复文字，变形文字，英文乱字符，杂乱背景，过度霓虹，低清晰度，水印，logo
```

## Notes

- 中文文字图建议标题短、字数少、放在前面。
- 正式发布仍建议人工复核文字。
