# 知识星球延伸帖提示词

## 资产信息

| 字段 | 内容 |
|------|------|
| 类型 | 内容创作 / 社群运营 |
| 适用场景 | 将公开内容延伸成知识星球资料、答疑或专题讲解 |
| 适用模型/平台 | ChatGPT / Claude / Gemini |
| 输入变量 | `{公开内容主题}`、`{用户问题}`、`{可提供资料}`、`{行动任务}` |
| 输出结果 | 星球帖子、资料清单、答疑引导 |
| 标签 | 知识星球, 社群, 资料库, 答疑 |
| 版本 | v1.0 |

## 标准版提示词

```text
请把这个公开内容主题扩展成一篇知识星球帖子。

公开内容主题：{公开内容主题}
用户常见问题：{用户问题}
可提供资料：{可提供资料}
希望用户完成的行动：{行动任务}

要求：
- 不重复公开视频/公众号的全部内容
- 提供更系统的补充、清单或操作步骤
- 适合星球成员收藏
- 结尾引导成员留言提问或提交作业

请输出：
1. 帖子标题
2. 帖子正文
3. 资料清单
4. 作业或行动任务
5. 答疑引导
```

## 增强版提示词

```text
你是「源力魔方」知识星球的内容运营。请将 `{公开内容主题}` 设计成一篇星球延伸帖。

定位：
- 公开视频负责扫盲
- 公众号负责图文沉淀
- 星球负责系统资料、专题补充和答疑陪跑

输出要求：
## 适合谁看
## 公开内容没有展开的部分
## 深度补充
## 可下载/可收藏资料
## 本周练习任务
## 常见问题
## 评论区互动引导
```

## 使用建议

适合把 B站和公众号内容做深一层，避免星球内容只是公开内容复制。
