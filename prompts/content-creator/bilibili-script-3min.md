# 3 分钟 B站 AI 工作流脚本

## 资产信息

| 字段 | 内容 |
|------|------|
| 类型 | 内容创作 |
| 适用场景 | 生成 3 分钟 AI 工作流扫盲 / ComfyUI 实操视频脚本 |
| 适用模型/平台 | ChatGPT / Claude / Gemini |
| 输入变量 | `{选题}`、`{目标读者}`、`{核心观点}`、`{案例}`、`{承接动作}` |
| 输出结果 | 3 分钟视频脚本、镜头提示、结尾引导 |
| 标签 | B站, 脚本, AI工作流, ComfyUI |
| 版本 | v1.0 |

## 标准版提示词

```text
请为 B站生成一条 3 分钟 AI 工作流扫盲视频脚本。

选题：{选题}
目标观众：{目标读者}
核心观点：{核心观点}
案例或演示：{案例}
承接动作：{承接动作}

结构要求：
1. 0-10 秒：痛点开场
2. 10-40 秒：解释一个关键概念
3. 40-140 秒：案例或界面演示
4. 140-165 秒：常见误区
5. 165-180 秒：总结、下期预告、关注理由

请输出：
- 分段脚本
- 每段画面建议
- 置顶评论要点
- 标题备选 5 个
```

## 增强版提示词

```text
你是「源力魔方」的 B站脚本策划，专门把 AI 工作流、ComfyUI、AI 生图和 AI 视频概念讲给新手听。

请围绕 `{选题}` 写一条 3 分钟视频脚本。

要求：
- 开场直接命中新手困惑，不讲空话
- 全片只讲清一个核心判断：{核心观点}
- 每 30 秒有一个信息推进点
- 能自然承接公众号图文、GitHub 资料和后续系列
- 不要用“最强、封神、颠覆”这类夸张词
- 不要把脚本写成文章，要适合口播

输出格式：
## 视频标题
## 目标观众
## 核心判断
## 分段脚本
## 画面/录屏建议
## 结尾关注理由
## 置顶评论草稿
```

## 使用建议

适合做系列视频的第一版脚本。拿到脚本后，建议再单独生成封面标题、视频简介和置顶评论。
