# 研究记录

本库不是复制公开 prompt，而是学习高质量提示词资料的结构，再转化为「源力魔方」自己的资产卡片。

## 参考方向

| 来源类型 | 学到什么 | 如何转化到本库 |
|------|------|------|
| OpenAI Prompt Engineering | 指令前置、上下文分隔、明确输出格式、少用模糊词 | 每张资产卡片都包含任务、变量、输出格式和使用建议 |
| Anthropic Prompt Engineering | 先定义成功标准，再迭代 prompt | 增加资产标准、版本记录和适用场景 |
| Awesome Prompts 类 GitHub 库 | 按任务分类、角色化 prompt 容易复用 | 但本库不堆角色，优先服务 AI 工作流场景 |
| PromptArchive / ComfyPrompt | 按模型、任务和示例结果检索 | 本库按内容创作、AI 生图、AI 视频、ComfyUI 分类 |
| FLUX Prompt Builder 类项目 | 镜头、光线、相机、风格预设很适合资产化 | 新增 FLUX 自然语言构建器和视觉类资产 |
| Seedance 2.0 公开资料 | 视频生成需要文本、图像、视频、音频等多模态参考 | 新增分镜、角色一致性、首尾帧和参考素材包资产 |

## 提炼原则

- 不追求模板数量，优先保留真实会用的工作流资产。
- 不照抄第三方 prompt，转成自己的变量、结构和使用说明。
- 不把提示词写成玄学咒语，尽量写清楚输入、输出、限制和失败原因。
- 所有资产都要能服务当前副业主线：B站、公众号、GitHub、RunningHub、知识星球。

## 下一步建议

- 为每张资产增加生成案例图或使用截图。
- 给每张资产补充 `tested` 状态和适用模型版本。
- 后续增加 RunningHub 工作流链接或示例。
- 从实际 B站视频和公众号文章中反向沉淀新资产。
