# AI 视频分镜表生成器

## 资产信息

| 字段 | 内容 |
|------|------|
| 类型 | AI 视频 |
| 适用场景 | 将短片想法、广告概念或产品故事拆成镜头表 |
| 适用模型/平台 | ChatGPT / Claude / Gemini / Seedance 前期策划 |
| 输入变量 | `{故事概念}`、`{片长}`、`{镜头数量}`、`{主角}`、`{风格}` |
| 输出结果 | 分镜表、镜头作用、生成风险 |
| 标签 | AI视频, 分镜, Seedance, 镜头脚本 |
| 版本 | v1.0 |

## 标准版提示词

```text
请把这个短片想法拆成 AI 视频分镜表。

故事概念：{故事概念}
片长：{片长}
镜头数量：{镜头数量}
主角：{主角}
视觉风格：{风格}

要求：
- 每个镜头只承担一个叙事功能
- 保持角色、服装、场景连续
- 标注景别、镜头运动、画面动作、情绪
- 标注每个镜头的 AI 生成风险

请输出 Markdown 表格。
```

## 增强版提示词

```text
你是 AI 视频分镜导演。请将 `{故事概念}` 拆成适合 Seedance / 图生视频工具执行的镜头表。

输出字段：
镜头编号、时长、叙事作用、景别、机位、镜头运动、画面动作、角色状态、场景连续性、光线色彩、生成风险、关键帧建议。

规则：
- 不要一镜头塞多个复杂动作
- 不要随意新增角色或场景
- 复杂镜头拆成多个短镜头
- 明确哪些镜头需要首尾帧
- 每个镜头都要能单独转成 prompt
```

## 使用建议

适合在生成 Seedance prompt 之前使用。先锁定分镜，再逐镜写 prompt，比直接写长 prompt 稳定。
