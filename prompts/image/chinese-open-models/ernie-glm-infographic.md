# ERNIE / GLM Infographic

适用模型：ERNIE-Image、GLM-Image、Qwen-Image

## Prompt

```text
生成一张中文信息图风格的 AI 教程配图，主题是“AI 生图模型怎么选”。画面分为三层：第一层是“快速出图”，第二层是“中文文字”，第三层是“图片编辑”。每层用简洁图标和卡片表达，不要生成大量小字。整体风格像清爽的知识型PPT页面，浅色背景，蓝灰配色，层级清晰，适合后期添加中文说明。
```

## Negative Prompt

```text
密密麻麻的小字，错别字，乱码，复杂表格，低质量，水印，logo，过度装饰
```

## Notes

- 信息图模型可以尝试短中文标签，但正式图文建议后期排版。
