# HunyuanImage Instruct Visual Edit

适用模型：HunyuanImage-3.0-Instruct、Qwen-Image-Edit、Step1X-Edit 类编辑模型

## Prompt

```text
请基于输入图片进行编辑：保持人物身份、脸部特征、服装和姿势不变，只将背景替换为一个简洁的现代 AI 工作室。背景包含柔和蓝白色灯光、干净桌面、轻微虚化的屏幕和节点流程线元素。整体风格真实、专业、适合 AI 教程封面。不要改变人物五官，不要改变衣服颜色，不要添加新人物，不要生成文字或 logo。
```

## Negative Prompt

```text
改变人物身份，换脸，换衣服，新增人物，乱码文字，logo，水印，过度赛博朋克，背景杂乱
```

## Notes

- 适合图像编辑模型：重点说清“保持什么不变”和“只修改什么”。
