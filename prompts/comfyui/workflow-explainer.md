# ComfyUI 工作流解释器

## 资产信息

| 字段 | 内容 |
|------|------|
| 类型 | ComfyUI / 教程 |
| 适用场景 | 将复杂工作流解释成新手能理解的输入、处理、输出结构 |
| 适用模型/平台 | ChatGPT / Codex / Claude |
| 输入变量 | `{工作流截图或节点列表}`、`{目标效果}`、`{用户水平}` |
| 输出结果 | 工作流结构解释、关键节点、学习建议 |
| 标签 | ComfyUI, 工作流, 节点, 教程 |
| 版本 | v1.0 |

## 标准版提示词

```text
请把这个 ComfyUI 工作流解释给新手听。

工作流截图或节点列表：
{工作流截图或节点列表}

目标效果：
{目标效果}

用户水平：
{用户水平}

请按这个结构输出：
1. 这个工作流输入什么
2. 中间大概分成哪几个模块
3. 最终输出什么
4. 新手最需要看懂的 5 个节点
5. 哪些参数前期不要乱动
6. 如果想改效果，优先改哪里
```

## 增强版提示词

```text
你是 ComfyUI 新手教程讲解员。请把复杂工作流拆成“输入 -> 处理 -> 输出”的结构。

要求：
- 不要逐个背节点名
- 先讲模块，再讲关键节点
- 对每个模块说明它解决什么问题
- 标注新手可改项和危险项
- 用类比帮助理解，但不要过度简化

输出：
## 工作流总览
## 输入区
## 模型区
## 控制区
## 采样生成区
## 后处理区
## 输出区
## 新手修改建议
```

## 使用建议

适合做公众号教程、B站脚本和知识星球答疑。比“节点百科”更适合新手入门。
