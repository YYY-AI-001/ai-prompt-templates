# 系列化标题包生成器

## 资产信息

| 字段 | 内容 |
|------|------|
| 类型 | 内容创作 |
| 适用场景 | 为 B站、公众号和知识星球生成一组选题标题 |
| 适用模型/平台 | ChatGPT / Claude / Gemini |
| 输入变量 | `{主题}`、`{平台}`、`{目标用户}`、`{内容系列}` |
| 输出结果 | 标题包、封面短标题、系列编号建议 |
| 标签 | 标题, B站, 公众号, 系列化 |
| 版本 | v1.0 |

## 标准版提示词

```text
请为 `{主题}` 生成一组适合 `{平台}` 的标题。

目标用户：{目标用户}
内容系列：{内容系列}
语气：清楚、实用、新手友好，不夸张

请输出：
1. 正式标题 10 个
2. 封面短标题 10 个
3. 适合系列化的标题格式 3 套
4. 不建议使用的标题方向
```

## 增强版提示词

```text
你是「源力魔方」的内容标题编辑。请围绕 `{主题}` 设计一组标题资产。

要求：
- 标题必须明确告诉读者能解决什么问题
- 可以有轻微好奇心，但不要标题党
- B站标题优先考虑搜索和点击
- 公众号标题优先考虑收藏和转发
- 知识星球标题优先考虑资料感和系统感
- 保持系列统一感

输出格式：
## B站标题
## 公众号标题
## 封面短标题
## 系列命名方案
## 标题关键词
## 避免用词
```

## 使用建议

适合每周选题会或视频发布前使用。建议保留标题版本，后续根据播放和收藏数据复盘。
