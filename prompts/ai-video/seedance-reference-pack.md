# Seedance 参考素材包整理提示词

## 资产信息

| 字段 | 内容 |
|------|------|
| 类型 | AI 视频 / 素材整理 |
| 适用场景 | 将文字、图片、视频、音频参考整理为 Seedance 生成输入 |
| 适用模型/平台 | Seedance 2.0 / 多模态视频生成工具 |
| 输入变量 | `{项目目标}`、`{参考图}`、`{参考视频}`、`{参考音频}`、`{分镜}` |
| 输出结果 | 参考素材清单、用途标注、生成顺序 |
| 标签 | Seedance, 多模态, 参考素材, AI视频 |
| 版本 | v1.0 |

## 标准版提示词

```text
请帮我整理一个 Seedance 2.0 视频生成参考素材包。

项目目标：{项目目标}
参考图：{参考图}
参考视频：{参考视频}
参考音频：{参考音频}
已有分镜：{分镜}

请输出：
1. 每个素材的用途
2. 哪些素材用于角色一致性
3. 哪些素材用于场景/风格参考
4. 哪些素材用于运动/节奏参考
5. 每个镜头建议使用哪些参考
6. 缺失素材清单
```

## 增强版提示词

```text
你是 AI 视频制片助理。请把用户提供的多模态素材整理成 Seedance 2.0 可执行参考包。

要求：
- 不要把所有参考都塞给每个镜头
- 区分角色参考、场景参考、风格参考、动作参考、音频节奏参考
- 给每个镜头推荐最小必要参考
- 标出可能互相冲突的素材
- 输出一个生成顺序
```

## 使用建议

适合在视频生成前整理素材。Seedance 2.0 支持多模态参考时，素材越清楚，越容易控制角色、镜头和节奏。
