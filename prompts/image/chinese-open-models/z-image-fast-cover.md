# Z-Image Fast Cover

适用模型：Z-Image-Turbo、LongCat-Image、FLUX schnell 类快速模型

## Prompt

```text
一张简洁科技风封面图，主题是“开源生图模型”。画面中心是三个半透明模型卡片，分别象征快速出图、中文文字、图片编辑。卡片之间用细线连接，背景为深色渐变，蓝白色柔和发光，画面干净、现代、有教程感。不要生成任何可读文字，不要 logo，不要复杂 UI。16:9 横版构图。
```

## Negative Prompt

```text
杂乱界面，乱码文字，真实品牌logo，过度霓虹，低质量，模糊，水印
```

## Notes

- 适合快速出封面草图，正式标题后期加。
