# 使用指南

## 快速使用

1. 找到与你任务接近的资产卡片。
2. 复制“标准版提示词”。
3. 替换变量，例如 `{主题}`、`{目标用户}`、`{风格}`。
4. 根据平台选择是否使用增强版。
5. 保存你修改后的版本，记录成新的资产。

## 推荐工作流

```text
粗糙想法
  -> 提示词资产卡片
  -> 标准版 prompt
  -> 生成结果
  -> 记录问题
  -> 迭代 v1.1
```

## 什么时候用标准版

- 日常生成
- 快速试方向
- 任务简单
- 输出要求不复杂

## 什么时候用增强版

- 要交付给别人
- 需要稳定格式
- 内容较复杂
- 希望减少模型自由发挥

## 如何扩展新资产

优先从真实任务里提炼：

- B站封面
- 公众号文章
- ComfyUI 排错
- AI 视频分镜
- 工作流说明
- 模型扫盲图文

不要为了凑数量添加泛泛 prompt。
