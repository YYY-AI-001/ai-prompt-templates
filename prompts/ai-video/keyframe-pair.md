# AI 视频首尾帧提示词

## 资产信息

| 字段 | 内容 |
|------|------|
| 类型 | AI 视频 / 关键帧 |
| 适用场景 | 为图生视频准备首帧和尾帧提示词 |
| 适用模型/平台 | Seedance / 图生视频 / 文生图模型 |
| 输入变量 | `{镜头目标}`、`{首帧状态}`、`{尾帧状态}`、`{角色锚点}`、`{场景}` |
| 输出结果 | 首帧 prompt、尾帧 prompt、运动说明 |
| 标签 | 首尾帧, AI视频, 图生视频, 分镜 |
| 版本 | v1.0 |

## 标准版提示词

```text
请为一个 AI 视频镜头设计首帧和尾帧生图提示词。

镜头目标：{镜头目标}
角色锚点：{角色锚点}
场景：{场景}
首帧状态：{首帧状态}
尾帧状态：{尾帧状态}

要求：
- 首帧和尾帧必须保持角色、服装、场景一致
- 只改变动作、位置、表情或镜头状态
- 输出中文说明和英文生图 prompt
```

## 增强版提示词

```text
你是 AI 视频关键帧设计师。请为 `{镜头目标}` 设计一组首尾帧。

输出：
## 连续性锚点
## 首帧画面描述
## 首帧 English Prompt
## 尾帧画面描述
## 尾帧 English Prompt
## 中间运动说明
## 生成风险

规则：
- 首尾帧不能像两个不同场景
- 不要改变角色身份和服装
- 尾帧变化要适合 3-5 秒视频完成
```

## 使用建议

适合图生视频镜头。首尾帧变化越小，视频越稳定；变化越大，越容易崩。
