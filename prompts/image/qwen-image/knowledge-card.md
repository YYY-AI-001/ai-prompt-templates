# Qwen-Image Knowledge Card

适用模型：Qwen-Image、GLM-Image、ERNIE-Image、LongCat-Image

## Prompt

```text
生成一张适合中文知识卡片的竖版图片，主题是“模型能力对比”。画面上方是清晰中文标题：“模型能力对比”。画面主体分成左右两栏：左侧用一个发光核心表示“视觉质量”，右侧用一个结构化模块表示“文字理解”。整体风格简洁、现代、适合AI科普，不要过多小字。留出足够空白方便后期添加说明。浅色背景，蓝灰配色，信息图风格。
```

## Negative Prompt

```text
大量乱码文字，错别字，小字密集，杂乱信息图，低质量，水印，logo，过度装饰
```

## Notes

- Qwen/GLM/ERNIE 这类模型更适合尝试中文短标题和信息图。
- 小字说明建议后期排版。
