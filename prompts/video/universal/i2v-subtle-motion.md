# Universal I2V Subtle Motion

适用模型：Wan I2V、HunyuanVideo I2V、CogVideoX I2V、LTX-Video

## Prompt

```text
Animate the reference image into a 4-second subtle cinematic shot. The subject remains in the same position and keeps the same identity, clothing, and facial features. Only add gentle breathing, a slight head turn toward the camera, soft blinking, and subtle movement in the background light. The camera slowly pushes in, smooth motion, realistic lighting, no new objects, no text.
```

## Negative Prompt

```text
face change, clothing change, new person, large pose change, distorted hands, flickering, sudden camera shake, warped background, subtitles, watermark, logo
```

## Notes

- 图生视频首选小动作，稳定性比大动作高。
