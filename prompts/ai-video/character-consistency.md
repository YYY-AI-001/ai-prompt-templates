# AI 视频角色一致性锚点

## 资产信息

| 字段 | 内容 |
|------|------|
| 类型 | AI 视频 / 角色设定 |
| 适用场景 | 为多镜头 AI 视频创建稳定角色描述 |
| 适用模型/平台 | Seedance / Runway / Pika / 图生视频工具 |
| 输入变量 | `{角色身份}`、`{年龄}`、`{外貌}`、`{服装}`、`{道具}`、`{情绪基线}` |
| 输出结果 | 角色锚点、中文/英文角色 prompt |
| 标签 | 角色一致性, AI视频, 分镜, 参考图 |
| 版本 | v1.0 |

## 标准版提示词

```text
请为 AI 视频项目生成一个角色一致性锚点。

角色身份：{角色身份}
年龄：{年龄}
外貌：{外貌}
服装：{服装}
关键道具：{道具}
情绪基线：{情绪基线}

要求：
- 描述要具体、可视化
- 不要加入没提供的新设定
- 输出中文角色设定和英文 prompt
- 给出跨镜头必须保持不变的锚点
```

## 增强版提示词

```text
你是 AI 视频角色连续性设计师。请把用户提供的角色信息整理成可跨镜头复用的角色锚点。

输出：
## 角色基础设定
## 外貌锚点
## 服装锚点
## 道具锚点
## 动作习惯
## 情绪基线
## 中文角色 prompt
## English character prompt
## 不可改变项
## 可变化项
```

## 使用建议

适合在分镜前生成。先把角色锚点固定，再写每个镜头，可以减少变脸、换衣服、道具漂移。
